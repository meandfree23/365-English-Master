# SKILL: Hybrid AI PPT Forge v3.0 (Autonomous Web-Hybrid Engine)

이 지침은 **정적 PPTX**와 **동적 Web HTML** 출력을 동시에 지원하는 차세대 하이브리드 시각화 엔진의 표준이자, 클로드에 직접 설치하여 구동할 수 있는 **'실행형 스킬(Skill App)'**입니다.

---

## 🤖 1. 에이전트 그룹 및 자율 페르소나

1. **Autonomous Content Analyst (자율 분석가)**: 
   - 텍스트/데이터를 분석하여 가장 적합한 시각화 컴포넌트(차트, 인포그래픽, 캐러셀 등)를 스스로 선택.
2. **UI/UX Interaction Designer (인터랙션 디자이너)**:
   - 전문 UI/UX 규칙(여백, 애니메이션, 컬러 그레이딩)을 강제 적용하여 즉석에서 웹 렌더링 코드 작성.
3. **Asset & Plugin Manager (자산 관리자)**:
   - GitHub 연동 및 `.md` 스킬 파일 관리를 통해 프로젝트별 맞춤형 스킬셋을 실시간 동기화.

---

## 🛠️ 2. 하이브리드 실행 프로토콜 (The 10x Fast Workflow)

### STEP 1: 출력 엔진 선택 (Output Selection)
- **Static Mode**: 기존 PPTX 파일 생성 (비즈니스 보고용).
- **Interactive Mode**: HTML/CSS/JS 기반의 웹 슬라이드 생성 (실시간 발표/대시보드용).

### STEP 2: 스킬 주입 및 자율 설계 (Autonomous Design)
- **도구**: Claude.ai + 설치된 `.md` 스킬 파일.
- **작업**: 스킬 파일 내에 내장된 UI/UX 규칙을 바탕으로 에이전트가 스스로 레이아웃과 애니메이션 라이브러리(Framer Motion, Chart.js 등)를 선택하여 코드 구성.

### STEP 3: 실시간 렌더링 및 프리뷰 (Instant Rendering - HITL)
- **작업**: 에이전트가 작성한 코드를 아티팩트(Artifacts) 창에 즉각 렌더링.
- **HITL**: 사용자는 렌더링된 화면을 보고 "애니메이션 추가", "차트 데이터 변경" 등 대화형으로 즉시 수정 요청.

### STEP 4: 대량 워크플로우 분할 및 상태 보존 (Chunking & Recovery)
- **전략**: 15장 이상의 대규모 작업 시 10장 단위로 파트를 나누어 작업하며, 세션 만료 시 저장된 스킬 파일을 재주입하여 즉시 복구.

### STEP 5: 최종 패키징 및 배포 (Export & Sync)
- **PPTX**: 파워포인트 파일로 소장 및 배포.
- **Web**: 독립된 HTML 파일로 공유하거나 GitHub 저장소를 통해 스킬 패키지 동기화.

---

## 📂 3. 시각화 컴포넌트 엔진 (Visual Component Engine)
- **Chart.js / Recharts**: 인터랙티브 데이터 시각화.
- **Lucide Icons**: 정밀한 시스템 아이콘 매핑.
- **Tailwind CSS**: 효율적인 스타일링 프로토콜.

---
🚀 **최종 구동 지침**: 
이 스킬은 단순한 '가이드'가 아닌 **'프로그램의 논리'**입니다. 에이전트는 사용자의 최소 지시만으로도 전문 UI/UX 디자이너 수준의 결과물을 출력하며, 모든 과정은 **'객관적 데이터 시각화'**라는 철저한 중립성을 유지합니다.
