# Professional Business & UI/UX Design Skill v3.0 (Assetization)

이 지침은 **기업 표준(Standard)**과 **현대적 UI/UX 디자인 시스템**을 결합하여 자산화시킨 최종 디자인 스킬입니다. 

---

## 🎨 1. 컬러 팔레트 & 브랜딩 (Professional Identity)

- **Primary (Deep Navy)**: `#0B2135` (텍스트 및 헤더용) - 중후한 신뢰감.
- **Secondary (Business Blue)**: `#004797` (포인트 및 데이터 시각화용) - 전문적인 느낌.
- **Accent (Corporate Orange)**: `#FF6B35` (강조/주의 문구용) - 시선 집중 및 데이터 반전.
- **Surface (Neutral Grey)**: `#F4F7FB` (슬라이드 배경 테마) - 휴계적 여백 확보.
- **Background (Pure White)**: `#FFFFFF` (본문 텍스트 영역 및 인포그래픽 영역).

### Theme Options (Responsive)
- **Light Mode**: 기본 기업 표준 화이트 테마.
- **Dark Mode**: `#0D1117` (Deep Grey) 배경 테마를 통해 강조 슬라이드 및 웹 렌더링 시 가독성 확보.

---

## 🖋️ 2. 타이포그래피 & 타이포 시각화 (Legibility Engine)

- **Header (Primary)**: `Pretendard Bold` (42pt+) - 명확한 메시지.
- **Sub-Header**: `Pretendard Medium` (28pt) - 계층 구조.
- **Governing Message**: `Pretendard Light` (14-16pt) - 슬라이드 상단 그레이바(`#F4F7FB`) 내 요약.
- **Web Typeface**: Inter / Roboto (웹 렌더링 시 대안).

---

## 📐 3. UI/UX 디자인 시스템 (Component-based)

### SPACING & GRID
- **Base Grid**: 8px 단위 그리드 시스템 강제.
- **Safe Zone**: 슬라이드 상단 80px(헤더), 하단 40px(푸터) 안전 구역 설정.

### INTERACTION RULES
- **Framer Motion**: 웹 슬라이드 생성 시 0.5s 이상의 전환 애니메이션 적용 금지 (속도감 최우선).
- **Chart Selection**: 데이터가 5개 이상일 경우 파이 차트 지양, 막대 차트 또는 히트맵 선호.

---

## 🤖 4. AI 생성 자율 프로토콜 (Autonomous Agent)

### 📄 스킬 파일 자산화 (.md Export)
"Generate a reusable .md skill file that includes:
- Color/Font/Layout rules. 
- UI/UX interaction logic (8px grid, Chart.js templates). 
- Professional Persona: Content-focused UI Designer."

### 📄 하이브리드 자동화 프라이빗 (Hybrid Prompt)
"Create an interactive slide presentation using HTML/JS/CSS. 
- Constraint: Follow the exact corporate style sheet. 
- Instruction: Use Lucide icons and Chart.js for data. Apply smooth fade-in animations for each component."

🚀 **디자인 철학**: 
이 스킬은 **'장식'보다 '기능'**을 디자인합니다. 모든 시각화 결과물은 8px 그리드와 데이터 위계에 따라 배치되며, 웹 환경과 PPTX 환경 모두에서 완벽한 일관성을 유지합니다.
